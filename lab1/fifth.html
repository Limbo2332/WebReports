Теорія

Тег <li> (англ. list item - елемент списку) створює елемент впорядкованого (нумерованого) або маркованого (ненумерованого) списку.

Тег <li> використовується у впорядкованих списках (<ol>), марковані списки (<ul>), а також в меню (<menu>).

Маркований (невпорядкований) список в HTML визначається тегом <ul> (Unordered List Element). Кожен елемент списку повинен починатися з тега <li> (скорочене від англійського list item - елемент списку).

Тег <ol> (агнл. ordered (enumerated) list - визначає впорядкований (пронумерований) список.

Атрибути <li>

Атрибут Опис атрибута
type Встановлює вид маркера нумерованного або маркованого списку
value Визначає номер елемента для нумерованого списку

Атрибути <ul>

Атрибут Опис атрибута
compact Вказує, що список має бути меншим ніж зазвичай
type Визначає як буде позначено кожен елемент списку

Атрибути <ol>

Атрибут Опис атрибута
compact Вказує, що список має бути меншим, ніж зазвичай
reversed Нумерація в списку буде по спадаючій (3,2,1)
start Задає число, з якого буде починатися нумерований список.
type Визначає вид маркера для використання в списку.
